## 시간 복잡도

빅오 표기법의 특징을 말하시오

```
시간 복잡도의 점근적 상한을 나타낸다. 
시간 복잡도 다항식의 최고차항만 계수 없이 취하여 사용한다. 
 	- 상수항은 무시한다. 
 	- 상대적으로 영향력이 작은 항은 무시한다. 
```



T(n) = T(n-1) + n, T(0) = 1 의 시간복잡도를 구하시오

```
T(n) = T(n-2) + n-1 + n
	 = T(n-3) + n-2 + n-1 + n
	 = T(n-k) + n-k+1 + n-k+2 + ⋯ + n
	 = T(0) + 1 + 2 + ⋯ + n  (n=k) 
	 = 1 + (n(1+n))/2
	 
	 최고차항은 n²/2
	 ∴ O(n²)
```
