## 시간 복잡도

빅오 표기법의 특징을 말하시오

```
시간 복잡도의 점근적 상한을 나타낸다. 
시간 복잡도 다항식의 최고차항만 계수 없이 취하여 사용한다. 
 	- 상수항은 무시한다. 
 	- 상대적으로 영향력이 작은 항은 무시한다. 
 	
#   
- 빅오표기법은 시간복잡도의 점근적 상한을 나타내는 표기법이다. 최악의 경우, 이정도의 시간이 걸린다는 의미로 해석할 수 있다.
- 빅오표기법은 데이터의 입력값이 충분이 크다고 가정하기 때문에 비교적 작은 항은 무시하고 최고차항만을 계수없이 나타낸다.
```

```
빅오 표기법은 시간 복잡도를 표기하는 대표적인 방법이며 시간 복잡도란 어떤 알고리즘을 수행하는데 걸리는 시간을 설명하는 계산 복잡도를 의미한다. 
시간 복잡도의 점근적 상한을 나타낸다. 
시간 복잡도 다항식의 최고차항만 계수 없이 취하여 사용한다. 
 	- 상수항은 무시한다. 
 	- 상대적으로 영향력이 작은 항은 무시한다. 
```

```
- 빅오표기법은 시간복잡도의 점근적 상한을 나타내는 표기법이다. 최악의 경우, 이정도의 시간이 걸린다는 의미로 해석할 수 있다.
- 빅 오 표기법은 데이터의 입력값이 충분이 크다고 가정하기 때문에 비교적 작은 항은 무시하고 최고차항만을 계수없이 나타낸다
```



T(n) = T(n-1) + n, T(0) = 1 의 시간복잡도를 구하시오

```
T(n) = T(n-2) + n-1 + n
	 = T(n-3) + n-2 + n-1 + n
	 = T(n-k) + n-k+1 + n-k+2 + ⋯ + n
	 = T(0) + 1 + 2 + ⋯ + n  (n=k) 
	 = 1 + (n(1+n))/2
	 
	 최고차항은 n²/2
	 ∴ O(n²)
```



```
빅오 표기법은 시간 복잡도를 표기하는 대표적인 방법이며 시간 복잡도란 어떤 알고리즘을 수행하는데 걸리는 시간을 설명하는 계산 복잡도를 의미한다. 
빅오 표기법은 데이터 입력값이 충분히 크다고 가정하고 있기 때문에 상수항 같은 사소한 부분은 무시하고 영향력이 가장 큰 최고차항을 계수없이 표기한다. 
O-표기(빅오표기)는 복잡도의 점근적 상한을 나타낸다. 
n의 증가에 따라 어느정도 시간이 걸리는지 보여준다. 
점근적 상한이기 때문에 제일 오래 걸리면 즉, 최악의 경우에는 이 정도의 시간이 걸릴 것이다라는 의미로 해석할 수 있다.
```





```
빅오 표기법은 시간 복잡도를 표기하는 대표적인 방법이며, 알고리즘을 수행하는데 걸리는 시간을 설명하는 계산 복잡도이다. 
시간 복잡도의 점근적 상한을 나타내는 표기법이며 최악의 경우 이 정도 시간이 걸린다는 의미
데이터의 입력값이 충분히 크다고 가정하기 때문에 비교적 작은 차수의 항은 무시하고 최고차항을 계수없이 나타낸다. 
```
